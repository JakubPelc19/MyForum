# MyForum (Work in Progress)

MyForum je jednoduchá, moderní a rychlá webová aplikace typu fórum, která umožní uživatelům zakládat témata, psát příspěvky a diskutovat v přehledném prostředí. Projekt je momentálně **ve vývoji**.

## Projekt je stále ve vývoji
Tento repozitář obsahuje nedokončenou verzi aplikace. Funkce, struktura projektu i UI se mohou rychle měnit.

## Tech Stack
- ASP.NET Core MVC
- .NET 10
- Entity Framework Core
- PostgreSQL
- HTML/CSS
- JavaScript
